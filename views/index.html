Welcome to Songs API Preview!

In this page you can test API in Postman using these endpoints:

GET

  • GET songs: /songs
  • GET song by id: /song/:id
  • GET songs by album: /songs/:album/album

POST

  • POST song with key "title" and "album": /songs

    • Available try at address bar:

  • POST song with params "title" and "album": /song/:title/:album

UPDATE

  • UPDATE song by id: /song/:id

    • Available try at address bar:

  • UPDATE song by id, title, album: /song/:id/:title/:album

DELETE

  • DELETE song by id: /song/:id

This API will send you feedback in JSON format
