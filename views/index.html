Welcome to Songs API Preview!

In this page you can test API in Postman using this endpoints:

  • GET songs: /songs
  • POST song with key "title" and "album": /songs (using Postman)
  • POST song with params "title" and "album": /song/:title/:album (from address bar)
  • UPDATE song by id: /song/:id (using Postman)
  • UPDATE song by id, title, album: /song/:id/:title/:album (from address bar)
  • DELETE song by id: /song/:id


This API will send you feedback in JSON format


PORT: 5000
